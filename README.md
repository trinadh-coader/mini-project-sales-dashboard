# Supermarket Sales & Profit Dashboard 📊

## Overview
This is my first mini-project as an AI & Data Science student. I wanted to practice extracting business insights from raw data, so I used a Supermarket dataset from Kaggle to build an interactive dashboard.

## Technical Skills Applied
* **Data Cleaning:** Handled missing values and formatted raw Kaggle data.
* **Data Aggregation:** Used Pivot Tables and advanced Excel logic to group daily transaction data into monthly/quarterly trends.
* **UI/UX Design:** Focused on the "data-ink ratio" by removing gridlines, standardizing the color palette, and creating a clean, app-like interface.
* **Interactive Elements:** Implemented slicers to allow dynamic filtering by Region, Segment, and Category.
